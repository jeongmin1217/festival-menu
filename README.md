## Functions
- 로그인 & 회원가입
- 유저/사장님 이용 공간 분리
- 사장님 가게 등록 기능
- 유저는 카테고리별 사진관을 한 눈에 보기 가능
- 유저/사장님 각자에 알맞은 CRUD 작업

## Architecture
- Front-end : Javascript, HTML, CSS, jQuery
- Back-end : PHP
- DB : MySQL
- Cloud & Hosting : 교수님 연구실 서버 대여
- Tools : jEdit, phpMyAdmin, VScode
